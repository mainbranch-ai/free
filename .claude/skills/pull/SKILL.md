# Pull

Pull latest updates from GitHub.

---

## What It Does

```bash
cd ~/Documents/GitHub/mb-free && git pull origin main
```

**Updated:** "Pulled latest. Changes: [list]"

**Current:** "Already up to date."

**Error:** "Couldn't pull: [error]. Try again later."
